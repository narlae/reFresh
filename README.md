## 📋 개요

> 프로젝트 명 : Only Fresh
>
- 다함께 주제 선정에 관한 논의가 이뤄지던 중에
이번 팀 프로젝트의 학습과 관련도가 높고
웹페이지 사용자로서 가장 친숙한 형태라고 볼 수 있는 이커머스가
모두의 공통적인 관심사로 선정되어 주제로 정하게 되었습니다.
